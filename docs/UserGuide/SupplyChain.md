# Supply Chain Modeling

The Supply Chain view allows you to design and monitor resource flows (Oxygen, Hydrogen, Power, etc.) using a node-based interface.

## ðŸ­ Designing Flows

1.  **Sources**: Extract resources (e.g., Regolith).
2.  **Processors**: Convert resources (e.g., Electrolysis: Water -> O2 + H2).
3.  **Sinks**: Store or consume resources (e.g., Fuel Tanks, Life Support).

Connect nodes to establish flow. The **Solver** calculates rates, pressures, and valid interactions in real-time.

> [!TIP]
> Use the **Details Panel** to tweak node parameters like "Efficiency" or "Power Output" on the fly and watch how it affects the entire grid.

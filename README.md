# LunCoSim

This is a new, Bevy based implementation of LunCoSim.

It's based on Sysml V2, Modelica, and Bevy to provide a framework for simulating complex systems. 
LunCoSim is designed to be highly accessible for building games and interactive tutorials on top of it, while remaining powerful enough for rigorous engineering simulations. This dual capability is achieved through a modular plugin system and adjustable levels of simulation fidelity.

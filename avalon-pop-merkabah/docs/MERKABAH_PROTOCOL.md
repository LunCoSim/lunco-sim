# MERKABAH qhttp:// Protocol Specification

## Introduction
The `qhttp://` protocol is a quantum-ready extension of HTTP designed for ASI/AGI orchestration within the MERKABAH framework.

## Endpoints

### `POST /qhttp/scan`
Initiates a spectral scan of a region to detect persistent order patterns.
**Payload**: `QuantumCommand`

### `POST /qhttp/inject`
Injects a harmonic signal (e.g., from Suno) into the global quantum mesh.
**Payload**: `HarmonicPayload`

### `POST /qhttp/verify`
Triggers a quantum consensus protocol (GHZ-based) to verify a detected anomaly.
**Payload**: `POPReport`

### `GET /qhttp/status`
Returns the current state of the global resonance field and network coherence.

### `POST /qhttp/alert`
Broadcasts a civilization-level alert through the quantum-entangled mesh.
